            CSS 인터랙션을 넘어,
            JavaScript까지 탐구하는 퍼블리셔

            안녕하세요. 저는 2년간 교육 콘텐츠 퍼블리싱을 경험하며 직관적인 인터페이스 구현에 집중해왔습니다. 특히 CSS와 SVG를 활용한 인터랙티브한 움직임을 구현하는데 많은 노력을 기울였습니다.

            웹 퍼블리싱은 저에게 새로운 도전이지만, 그동안의 경험과 학습을 바탕으로 성장해 나가고자 합니다. 부드러운 동적 효과를 구현하고, 중복되는 코드를 줄여 효율적인 UI 설계를 할 수 있도록 신경 쓰겠습니다.

            최신 기술을 익히며 사용자에게 더 나은 경험을 제공하는 웹사이트를 만들고, 변화하는 환경에도 빠르게 적응하는 퍼블리셔로 성장하겠습니다.
